# Speedcode gen project

This project is to experiment with speed code generators on the C64. 

## Building the project

```bash
# make
```

## Run the project is vice

Omit the '.asm' extension 
```bash
# make run-<file name> 
```

## Debug 

Omit the '.asm' extension 
```bash
# make debug-<file name>
```

## Example

```bash
# make run-sinewave
```
